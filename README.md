

# Лабиринт сокровищ
Добро пожаловать в **Лабиринт сокровищ** — текстовую приключенческую игру на Python!
В этой игре вы — отважный исследователь, блуждающий по древнему лабиринту, полному хитрых коридоров, скрытых сокровищ и загадок. Ваша задача — найти главный приз, используя только логику и код.
> Игра полностью на чистом Python, без внешних библиотек (кроме стандартных). Подходит для обучения основам программирования, Git и Poetry.

---

## Установка
Чтобы установить проект, выполните следующие шаги:
### 1. Клонируйте репозиторий
1. Используйте bash
git clone https://github.com/ksenofkv/project1_Ksenofontov_Konstantin_M25-555
cd ваш-репозиторий
###2. Установите зависимости
Используйте Poetry: 'poetry install' или используйте Makefile: 'make install'

## Запуск игры
После установки запустите игру командой: 'poetry run project' или 'make project'

## Как играть?
go <direction> — перейти в направлении (north/south/east/west)
look — осмотреть текущую комнату
take <item> — поднять предмет
use <item> — использовать предмет из инвентаря
inventory — показать инвентарь
solve — попытаться решить загадку в комнате
quit — выйти из игры
help — показать это сообщение
